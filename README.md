# Customer Churn & Retention Analysis

## Project Overview
This project analyzes customer churn patterns to identify high-risk customers and retention opportunities.

## Tools Used
- Excel (Data cleaning, feature engineering)
- Tableau Public (Dashboard & visualization)

## Key Analysis
- Overall churn rate (KPI)
- Churn by contract type
- Churn by tenure group
- Impact of monthly charges on churn
- Churn by payment method

## Business Insights
- Month-to-month contracts show the highest churn

- Higher monthly charges increase churn probability
